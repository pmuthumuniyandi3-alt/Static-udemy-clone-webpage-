# Static-udemy-clone-webpage-
Udemy Clone Website

A simple Udemy Clone website built using HTML5 and CSS3. This project recreates the basic layout and design of the Udemy homepage, including a navigation bar, course categories, featured courses, and responsive styling. It was developed to practice front-end web development fundamentals such as page structure, styling, layouts, and responsive design.

Features
Clean and modern UI inspired by Udemy
Navigation bar with menu options
Course cards with images and details
Categories and featured sections
Responsive design using CSS
Built entirely with HTML and CSS (no frameworks)
Technologies Used
HTML5
CSS3
Purpose

This project was created as a learning exercise to improve my understanding of:

Semantic HTML structure
CSS styling and layouts
Flexbox/Grid concepts
Responsive web design principles
Front-end development best practices
Future Improvements
Add JavaScript for interactivity
Implement course search functionality
Create individual course pages
Add user authentication and backend integration
